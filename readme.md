# News

A simple & configurable site to display the latest news

## Getting started

* [Npm](https://nodejs.org)

```shell
npm i
```

## Commands

### Bundle with watch on dev environment

`yarn run dev`

### Bundle to production

`yarn run prod`

### Bundle & release

`yarn run release`

### Lint your code

`yarn run lint`
